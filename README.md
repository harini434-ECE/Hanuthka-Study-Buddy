# 🎓 Hanuthka’s Study Planner

Hanuthka’s Study Planner is an AI-powered web application designed to help students plan their studies, stay consistent, and improve learning efficiency through a personalized approach.

---

## 🚀 Live Demo

👉 https://hanuthka-study-buddy.lovable.app

---

## 💡 Overview

Many students struggle with:

* managing their study schedule
* staying consistent
* tracking their progress

This project solves these problems by combining structured planning with AI-based assistance.

---

## ✨ Features

* 📚 **Study Plan Generator**
  Create structured study plans based on exam dates and topics.

* 📊 **Progress Tracking**
  Track completed topics, daily performance, and study consistency.

* 🧠 **AI Coach**
  Get instant explanations, guidance, and motivation.

* 🔁 **Memory-Based AI (Hindsight)**
  The system remembers previous interactions and adapts responses to provide personalized support.

* 📝 **Quiz Generator**
  Generate quizzes from completed topics to improve revision.

---

## 🧠 Key Concept

Unlike traditional AI tools, this system uses a **memory-based approach**.

This allows the application to:

* remember user progress
* identify weak areas
* provide adaptive suggestions

This makes the experience more personalized and effective.
